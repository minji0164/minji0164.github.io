안녕하세요. 제 포트폴리오를 봐 주셔서 감사합니다.😊


저는 신입 웹 디자이너, 웹 퍼블리셔를 꿈꾸는 강민지입니다. 웹 개발의 역량을 키우기 위해 UIUX 웹앱디자인 과정에 열정적으로 참여하여 5개월간 성장의 시간을 가졌으며 현재는 부족한 자바스크립트 실력을 향상시키기 위해 꾸준히 공부하고 있습니다.🤓


제가 제일 자신 있게 할 수 있는 것은 시맨틱 태그를 이용한 코드 구성과 SCSS를 활용한 유지 보수가 용이한 스타일 시트를 작성하는 것입니다.


웹 디자이너이자 웹 퍼블리셔로서, 꾸준히 새로운 기술을 습득하고 트렌드를 읽는 능력이 가장 중요하다고 생각합니다. 저는 평소 독서를 통해 트렌드를 파악하고, 새로 등장하는 사이트와 블로그를 통해 다양한 최신 정보를 접하고 있습니다. 또한, 온라인 자료와 영상 강의, GitHub 및 CodePen과 같은 개발자 커뮤니티를 활용하여 부족한 기술을 보완하고 새로운 기술을 학습하고 있습니다.


이처럼 지속적인 학습과 경험을 통해, 사용자에게 최적의 웹 경험을 제공하고, 더욱 발전하는 웹 디자이너이자 퍼블리셔로 성장하고자 합니다.😎
